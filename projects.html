Neovim Configuration

An extensive and comprehensive Neovim configuration made to serve as a code editor for a variety of languages including Python, Java, C/C++, HTML, CSS, and JavaScript. It comes complete with features like LSPs and VSCode like snippets. It is easily extendable for users to add capabilties.

Immersive Camera for Witcher 3

An update and rework of Immersive Camera with support for Witcher 3's next generation version. It uses WitcherScript, a derivative of ActionScript 3, and changes much of the vanilla camera system in game.
